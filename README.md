# AI Medical Assistant using Endee

## Overview

This project is a simple AI-powered medical assistant that uses semantic search and Retrieval-Augmented Generation (RAG).

## Features

- Symptom-based query answering
- Medicine information retrieval
- AI-based responses

## Tech Stack

- Python
- Streamlit
- Endee Vector Database

## How it works

1. Medical data is stored
2. Converted into embeddings
3. Retrieved based on query
4. Response generated

## How to run

```bash
streamlit run app.py
```
